606 คลังข้อมูลและระบบสนับสนุนการตัดสินใจสำหรับการบริหารงานสินเชื่อเงินกู้ธุรกิจขนาดใหญ่ของธนาคารพาณิชย์ ธรรมรัตน์ เฟื่องฟุ้ง Microsoft SQL Server 2016, Tableau 10.1 โครงการอิมพลีเมนท์ เอสเอพี อีซีซี 6.0:โมดูลการซ่อมบำรุงโรงงานของธุรกิจผลิตกระแสไฟฟ้า พิริยะ เชียงกูล Oracle11g, SAP ECC 6.0 คลังข้อมูลและระบบสนับสนุนการตัดสินใจของร้านเบเกอรีและเครื่องดื่ม รวิวรรณ พิเชฐวีรชัย MySQL, Tableau 10.1 คลังข้อมูลและระบบสนับสนุนการตัดสินใจของธุรกิจโรงกลั่นเอทานอลจากมันสำปะหลัง รุจธิชัย ฉลองรัตนสกุล Microsoft SQL Server 2012, Tableau 10.1 ระบบเฝ้าระวังการวิเคราะห์ข้อมูลของธุรกิจให้บริการเฝ้าระวังยานพาหนะขนส่งสินค้า กฤษณะ จันลา Maria DB, PHP การจินตทัศน์ข้อมูลเพื่อการวิเคราะห์พฤติกรรมของผู้โดยสารและผู้เยี่ยมเยือนในสนามบิน วุฒิศักดิ์ อินปนาม MySQL,PHP, IBM Watson Analytics ระบบวิเคราะห์ข้อมูลเพื่อสนับสนุนการตัดสินใจกำหนดราคาธุรกิจอาหารสัตว์บก ณัฐสิริ ตันติขจร Microsoft SQL Server 2016, Power BI
607 คลังข้อมูลและระบบสนับสนุนการตัดสินใจของโรงงานผลิตยา ฐาปนีย คล้ายวงษ์ Microsoft SQL Server 2016, Tableau 10.4 โครงการอิมพลีเมนท์ ไอทอป 2.3 :ตามกรอบการดำเนินงานไอทิล 2011ของธุรกิจให้บริการดูแลระบบเทคโนโลยีสารสนเทศมนูญธรรม สุวรรณศักดิ์ MySQL - Maria DB โครงการอิมพลีเมนท์ เวิร์คเดย์ 27 :โมดูลการบริหารจัดการคนเก่งและผลการปฎิบัติงานของธุรกิจให้บริการที่ปรึกษาและพัฒนาระบบทางด้านเทคโนโลยีสารสนเทศ โชติรส งามชูเกียติ NoSQL Server 2016, Workday Cloud คลังข้อมูลและระบบสนับสนุนการตัดสินใจของธุรกิจผลิตอาหารสัตว์ สุพัตรา เตชนะรุ่งโรจน์ Microsoft SQL Server 2016, Tableau 10 คลังข้อมูลและระบบสนับสนุนการตัดสินใจของกลุ่มธุรกิจจำหน่ายสินค้าซีเมนต์และวัสดุก่อสร้างอลิษา กิ่งไพบูลย์ Microsoft SQL Server 2016, Power BI โครงการอิมพลีเมนท์ เอสเอพี อีซีซี 6.0:โมดูลการบริหารจัดการวัตถดิบของธุรกิจจำหน่ายตู้แช่สินค้า กฤช งามโชคชัยเจริญ Oracle 11g, SAP ECC 6.0 โครงการอิมพลีเมนท์ เอสเอพี อีซีซี6.0 :โมดูลการบริหารจัดการสินค้าคงคลังสำหรับธุรกิจจัดจำหน่ายอุปกรณ์การแพทย์ ณัฐบดี ศรีประสงค์ Oracle 11g, SAP ECC 6.0
608 คลังข้อมูลและระบบสนับสนุนการตัดสินใจสำหรับการให้บริการกระจายสินค้าของธุรกิจขนส่ง วิศรุษ บวรรัตนปราณ Microsoft SQL Server 2016, Qlik View 12.10 คลังข้อมูลและระบบสนับสนุนการตัดสินใจสำหรับการจัดการการให้บริการผ่านเครื่องรับจ่ายเงินอัตโนมัติของธุรกิจธนาคารพาณิชย์ ปภัสสร ศิริเต็มกุล Microsoft SQL Server 2016, Power BI คลังข้อมูลและระบบสนับสนุนการตัดสินใจสำหรับการจัดการปัญหาด้านแอพพลิเคชันของธุรกิจยานยนต์ นิชชา ปริงหาดยาย Microsoft SQL Server 2012, Tableau 10.4 คลังข้อมูลและระบบสนับสนุนการตัดสินใจสำหรับธุรกิจขายอะไหล่รถยนต์ บุญเอก สถิตเทพบัญชา Microsoft SQL Server 2017, Tableau 10.4 คลังข้อมูลและระบบสนับสนุนการตัดสินใจสำหรับการจัดการเหตุขัดข้องของธุรกิจให้บริการด้านเทคโนโลยีสารสนเทศ ดวงพร ตรัยสิงห์พิทักษ์ Microsoft SQL Server 2016, Power BI ระบบสารสนเทศเพื่อการจัดการงานสั่งทำของธุรกิจผลิตเสื้อผ้า วิทวัส วิทยาพิพัฒน์ Maria DB 10.1.26, Tableau 10.4 คลังข้อมูลและระบบสนับสนุนการตัดสินใจสำหรับการบริหารผลิตภัณฑ์เงินฝากของธนาคารพาณิชย์ ชมพูนุท วิสารทเวคิน Microsoft SQL Server 2012, Tableau 10.3
